# useTimeout

Advanced timeout management with pause, resume, and clear capabilities.

## Usage

```jsx
import { useTimeout } from "@motif-ui/react";

const AutoSaveForm = () => {
  const saveData = () => {
    console.log("Auto-saving data...");
  };

  const { start, pause, clear } = useTimeout(saveData, 5000);

  return (
    <div>
      <input onFocus={pause} onBlur={start} />
      <button onClick={start}>Start Auto-save</button>
      <button onClick={pause}>Pause Auto-save</button>
      <button onClick={clear}>Cancel Auto-save</button>
    </div>
  );
};
```

## Features

- **Pause/Resume**: Temporarily pause the timeout without losing progress
- **Dynamic Callback**: Automatically fires the callback function
- **Auto-clear**: Automatically clears timeout after execution
