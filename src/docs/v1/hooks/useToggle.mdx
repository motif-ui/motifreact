# useToggle

Manages visibility states with optional animation support.

## Basic Usage

```jsx
import { useToggle } from "@motif-ui/react";

const Sidebar = () => {
  const { visible, show, hide, toggle } = useToggle(false);

  return (
    <>
      <button onClick={toggle}>Toggle Sidebar</button>
      {visible && <div>Sidebar content</div>}
    </>
  );
};
```

## With Animation

```jsx
import { useToggle } from "@motif-ui/react";

const AnimatedModal = () => {
  const { visible, toggleState, show, hide } = useToggle(false, 300);

  return (
    <>
      <button onClick={show}>Open Modal</button>
      {(visible || toggleState) && (
        <div className={toggleState === "hiding" ? "fade-out" : "fade-in"}>
          Modal content
          <button onClick={hide}>Close</button>
        </div>
      )}
    </>
  );
};
```

## Animation Pattern

When `toggleTime` is provided:

1. `show()` sets `toggleState` to `"showing"` and `visible` to `true`
2. After `toggleTime` ms, `toggleState` becomes `undefined`
3. `hide()` sets `toggleState` to `"hiding"` and `visible` to `false`
4. After `toggleTime` ms, `toggleState` becomes `undefined`

This allows you to:

- Keep the element mounted during hide animation
- Apply different CSS classes based on `toggleState`
- Control when to unmount the element
