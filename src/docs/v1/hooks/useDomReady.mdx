# useDomReady

A simple hook that returns `true` when the DOM is fully loaded and ready.

## Usage

```jsx
import { useDomReady } from "@motif-ui/react";

const MyComponent = () => {
  const domReady = useDomReady();

  if (!domReady) {
    return <div>Loading...</div>;
  }

  return <div>DOM is ready!</div>;
};
```

## Use Cases

- Preventing hydration mismatches in SSR applications
- Ensuring DOM-dependent operations run safely
- Conditional rendering based on DOM readiness
