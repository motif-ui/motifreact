# useScreenSize

Tracks the current window dimensions and updates on resize.

## Usage

```jsx
import { useScreenSize } from "@motif-ui/react";

const ResponsiveComponent = () => {
  const { width, height } = useScreenSize();

  return (
    <div>
      <p>Window width: {width}px</p>
      <p>Window height: {height}px</p>
      {width < 768 && <p>Mobile view</p>}
    </div>
  );
};
```

## Use Cases

- Responsive layout adjustments
- Conditional rendering based on screen size
- Dynamic component sizing

## Note

This hook returns `{ width: 0, height: 0 }` during SSR. Make sure to handle this case in your components.
