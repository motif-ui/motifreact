# useTimeout

Durdurma, devam ettirme ve temizleme özellikleriyle gelişmiş zaman aşımı (timeout) yönetimi.

## Kullanım

```jsx
import { useTimeout } from "@motif-ui/react";

const AutoSaveForm = () => {
  const saveData = () => {
    console.log("Veriler otomatik kaydediliyor...");
  };

  const { start, pause, clear } = useTimeout(saveData, 5000);

  return (
    <div>
      <input onFocus={pause} onBlur={start} />
      <button onClick={start}>Otomatik Kaydetmeyi Başlat</button>
      <button onClick={pause}>Otomatik Kaydetmeyi Duraklat</button>
      <button onClick={clear}>Otomatik Kaydetmeyi İptal Et</button>
    </div>
  );
};
```

## Özellikler

- **Durdur/Devam Ettir**: Zaman aşımını (timeout) ilerlemeyi kaybetmeden geçici olarak duraklatın
- **Dinamik Geri Arama (Callback)**: Geri arama fonksiyonunu otomatik olarak çalıştırır
- **Otomatik Temizleme**: Çalıştırma sonrası zaman aşımını (timeout) otomatik olarak temizler
