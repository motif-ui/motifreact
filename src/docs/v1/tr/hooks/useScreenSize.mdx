# useScreenSize

Mevcut pencere boyutlarını takip eder ve yeniden boyutlandırmada günceller.

## Kullanım

```jsx
import { useScreenSize } from "@motif-ui/react";

const ResponsiveComponent = () => {
  const { width, height } = useScreenSize();

  return (
    <div>
      <p>Window width: {width}px</p>
      <p>Window height: {height}px</p>
      {width < 768 && <p>Mobile view</p>}
    </div>
  );
};
```

## Kullanım Alanları

- Duyarlı düzen ayarlamaları
- Ekran boyutuna göre koşullu render etme
- Dinamik bileşen boyutlandırma

## Not

Bu hook, SSR sırasında `{ width: 0, height: 0 }` döndürür. Bileşenlerinizde bu durumu ele aldığınızdan emin olun.
