# useToggle

İsteğe bağlı animasyon desteğiyle görünürlük durumlarını yönetir.

## Temel Kullanım

```jsx
import { useToggle } from "@motif-ui/react";

const Sidebar = () => {
  const { visible, show, hide, toggle } = useToggle(false);

  return (
    <>
      <button onClick={toggle}>Toggle Sidebar</button>
      {visible && <div>Sidebar content</div>}
    </>
  );
};
```

## Animasyonlu Kullanım

```jsx
import { useToggle } from "@motif-ui/react";

const AnimatedModal = () => {
  const { visible, toggleState, show, hide } = useToggle(false, 300);

  return (
    <>
      <button onClick={show}>Modali Aç</button>
      {(visible || toggleState) && (
        <div className={toggleState === "hiding" ? "fade-out" : "fade-in"}>
          Modal içeriği
          <button onClick={hide}>Kapat</button>
        </div>
      )}
    </>
  );
};
```

## Animasyon Deseni

`toggleTime` sağlandığında:

1. `show()` `toggleState`'i `"showing"` olarak ve `visible`'ı `true` olarak ayarlar.
2. `toggleTime` ms sonra, `toggleState` `undefined` olur.
3. `hide()` `toggleState`'i `"hiding"` olarak ve `visible`'ı `false` olarak ayarlar.
4. `toggleTime` ms sonra, `toggleState` `undefined` olur.

Bu size şunları sağlar:

- Gizleme animasyonu sırasında öğeyi bağlı tutma
- `toggleState`'e göre farklı CSS sınıfları uygulama
- Öğeyi ne zaman ayıracağınızı kontrol etme
