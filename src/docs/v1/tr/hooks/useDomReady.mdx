# useDomReady

DOM tamamen yüklendiğinde ve hazır olduğunda `true` döndüren basit bir kanca.

## Kullanım

```jsx
import { useDomReady } from "@motif-ui/react";

const MyComponent = () => {
  const domReady = useDomReady();

  if (!domReady) {
    return <div>Loading...</div>;
  }

  return <div>DOM is ready!</div>;
};
```
