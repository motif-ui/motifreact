# Temalar

Motif UI ekibi tarafından oluşturulmuş ve kullanıma hazır temalar bulunmaktadır. Sayfanın **sağ üst köşesinde** bulunan `tema değiştirme düğmesini` kullanarak birini seçebilir ve aşağıdaki sergide tüm bileşenlerin bu temayla nasıl göründüğünün canlı önizlemesini görebilirsiniz.

> **Not:** Kendi temanızı oluşturmak için lütfen <a href="../?path=/docs/customization-theming--docs">**Customization > Theming**</a> bölümüne bakın.
