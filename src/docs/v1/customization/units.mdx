# Units

<br />

## `REM` as the Preferred Measurement Unit

All components utilize `rem` units for defining spatial properties such as margin, padding, and width. By default,
**1rem** equals **16px**, following the standard browser configuration. This approach ensures consistent and scalable
sizing that adapts to the font-size defined on the html or :root element.

Below is an example demonstrating how adjusting the root font-size dynamically affects the sizing of components:

<div style={{ display: "flex", flexDirection: "column", padding: 40, gap: 12, margin: "0 auto", maxWidth: 400 }}>
  <Slider start={4} end={34} value={16} onChange={val => document.documentElement.setAttribute("style", `font-size: ${val}px;`)} />
  <PinCode value={["P", "I", "N"]} size="lg">
    <PinCode.Item />
    <PinCode.Item />
    <PinCode.Item />
  </PinCode>
</div>
