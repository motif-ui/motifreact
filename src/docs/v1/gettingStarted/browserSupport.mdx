# Browser Support

Motif UI supports the latest versions of all major browsers, including `Chrome`, `Safari`, `Firefox`, `Edge`, and `Opera`.
We actively track browser updates to maintain seamless compatibility, eliminate the need for polyfills, and ensure
optimal performance across all supported environments. A **Browserslist** configuration is also used in our build
process to guarantee consistent and up-to-date browser coverage.
