# Supported Components

The Form component supports a wide variety of input types stated in this page. In order to see a fully working example
of a form using these components, please refer to [Components/Form](?path=/docs/components-form--docs) page.

### Checkbox [[↗]](?path=/docs/components-checkbox--docs)

```jsx
<Form.FieldGroup name="permissions" label="Permissions">
  <Checkbox name="read" label="Read" />
  <Checkbox name="write" label="Write" />
  <Checkbox name="delete" label="Delete" />
</Form.FieldGroup>
```

### InputDate [[↗]](?path=/docs/components-inputdate--docs)

```jsx
<Form.Field name="birthdate" label="Birth Date">
  <InputDate />
</Form.Field>
```

### InputDateTime [[↗]](?path=/docs/components-inputdatetime--docs)

```jsx

  <InputDateTime />
</Form.Field>
```

### InputText [[↗]](?path=/docs/components-inputtext--docs)


### InputTime [[↗]](?path=/docs/components-inputtime--docs)


### InputPassword [[↗]](?path=/docs/components-inputpassword--docs)

### PinCode [[↗]](?path=/docs/components-pincode--docs)

```jsx
<Form.Field name="code" label="Verification Code">
  <PinCode>
    <PinCodeItem />
    <PinCodeItem />
    <PinCodeItem />
    <PinCodeItem />
  </PinCode>
</Form.Field>
```

### RadioGroup [[↗]](?path=/docs/components-radiogroup--docs)

```jsx
<Form.Field name="gender" label="Gender">
  <RadioGroup name="gender">
    <Radio value="male" label="Male" />
    <Radio value="female" label="Female" />
  </RadioGroup>
</Form.Field>
```

### Select [[↗]](?path=/docs/components-select--docs)

```jsx
<Form.Field name="interests" label="Interests">
  <Select
    multiple
    data={[
      { label: "Technology", value: "tech" },
      { label: "Art", value: "art" },
      { label: "Sports", value: "sports" }
    ]}
  />
</Form.Field>
```
